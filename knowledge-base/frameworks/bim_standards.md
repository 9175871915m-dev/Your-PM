# BIM — Building Information Modelling

## What is BIM?
Building Information Modelling (BIM) is a process for creating and managing information on a construction project across the project lifecycle. At the centre of BIM is an intelligent 3D model that is used as the central source of project information.

## BIM Levels (UK)
- **BIM Level 0:** 2D CAD drawings, no collaboration
- **BIM Level 1:** Mix of 3D CAD and 2D drawings, some collaboration
- **BIM Level 2 (UK Mandate):** Collaborative federated models, all parties share data in agreed format (IFC). Mandated on UK Government projects since April 2016.
- **BIM Level 3 (ISO 19650):** Fully integrated single model — future direction

## ISO 19650 Standard (current standard)
- Replaces PAS 1192
- Covers BIM information management for design, construction, and operation
- Two key parts: ISO 19650-1 (concepts), ISO 19650-2 (delivery phase)
- Introduces the **Common Data Environment (CDE)**

## Key BIM Roles
- **BIM Manager** — oversees BIM implementation strategy
- **BIM Coordinator** — manages model federation and clash detection
- **BIM Author** — produces the model (Revit, ArchiCAD, etc.)
- **Information Manager** — manages the CDE and data flow

## BIM Information Requirements
- **EIR (Employer's Information Requirements)** — client's BIM requirements
- **BEP (BIM Execution Plan)** — contractor's response to EIR, explains how BIM will be delivered
- **MIDP (Master Information Delivery Plan)** — schedule for all information deliverables
- **TIDP (Task Information Delivery Plan)** — individual consultant's delivery plan

## Common Data Environment (CDE)
A shared digital environment for managing project information:
- One central repository for all project models and documents
- Controlled workflow: Work in Progress → Shared → Published → Archived
- Tools: Autodesk Construction Cloud (ACC), Aconex, BIM 360, Procore, ProjectWise

## BIM Uses in Construction
1. **Clash Detection** — find and resolve MEP/structural clashes pre-construction
2. **4D BIM** — link model to programme (Navisworks TimeLiner)
3. **5D BIM** — link model to cost/quantities
4. **6D BIM** — sustainability analysis
5. **7D BIM** — facility management / asset management

## BIM on MEP Projects
- MEP modelling in Revit MEP — coordinated with architectural and structural models
- Clash detection between ductwork, pipework, electrical containment, and structure
- Spool drawings generated from model for off-site fabrication
- As-built model updated throughout — record model handed to FM team

## BIM Benefits
- Reduces RFIs and site queries by 30–60%
- Reduces rework through clash detection
- Improves programme certainty
- Enables accurate quantity take-off
- Supports FM handover with rich asset data
